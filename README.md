# Haim Bargig - Cinematographer Portfolio

## Adding Your Images & Videos

### 1. Hero Cover Image
- Replace `images/hero-cover.jpg` with your best cinematography still
- Recommended size: 1920x1080px or larger
- High contrast image works best with white text overlay

### 2. Portfolio Gallery Images
Upload your best cinematography stills to the `images/` folder:
- `portfolio-1.jpg` through `portfolio-6.jpg`
- Recommended size: 1200px wide minimum
- High-quality JPGs showcasing your cinematography work

### 3. Video Reel
- Upload your demo reel to **YouTube** or **Vimeo**
- Get the embed code from the platform
- Replace the placeholder iframe src URLs in index.html

### 4. Contact Information
Update these in index.html:
- Email address (line ~350)
- Add your actual reel link
- Update any other contact details

## File Structure
```
portfolio/
├── index.html          # Main website file
├── images/             # Your images folder
│   ├── hero-cover.jpg  # Main hero image
│   ├── portfolio-1.jpg # Gallery image 1
│   ├── portfolio-2.jpg # Gallery image 2
│   └── ...             # More gallery images
└── README.md          # This file
```

## Deployment
The site automatically deploys to GitHub Pages when you push changes.
Live URL: https://bargigh.github.io/portfolio/

## Customization
- All images can be replaced by uploading to the `images/` folder
- Video embeds can be updated in the HTML
- Colors and styling can be modified in the CSS section
